## Install

Make sure that you have the Gatsby CLI program installed:
```sh
npm install --global gatsby-cli
```

And clone this repo
```sh
git clone https://github.com/gouravsood/new-forming-app.git
```

Then you can run it by:
```sh
gatsby develop
```
